# Tephra Studios Website

Fertige Next.js Website für Vercel.

## Lokal starten
```bash
npm install
npm run dev
```

## Auf Vercel veröffentlichen
1. Ordner zu GitHub hochladen
2. Bei vercel.com einloggen
3. "Add New Project"
4. GitHub Repository auswählen
5. Deploy klicken

Kostenlose URL sieht dann z. B. so aus:
tephra-studios.vercel.app
